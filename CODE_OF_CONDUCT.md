# Code of Conduct

## Our Pledge

We want GPT Knowledge Builder to be a professional, respectful project for users and contributors of all backgrounds.

## Our Standards

Examples of behavior that contributes to a positive environment:

- Using respectful, precise language
- Giving and receiving constructive feedback
- Focusing on technical substance instead of personal attacks
- Respecting different levels of experience

Examples of unacceptable behavior:

- Harassment or discrimination
- Trolling, insulting, or derogatory comments
- Publishing private information without permission
- Disruptive conduct in issues, pull requests, or discussions

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that violate this Code of Conduct.

## Reporting

Report unacceptable behavior through the repository security or contact channel once it is configured for the public repo. Until then, open a private security report if the issue overlaps with abuse, privacy, or security.

## Scope

This Code of Conduct applies within all project spaces and when an individual is officially representing the project.
